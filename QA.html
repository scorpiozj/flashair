Master Code

The master code is a 12-digit hexadecimal number which is used as a password to set the SSID (APPSSID) and network security key (APPNETWORKKEY). As FlashAir does not have API to read master code, although Flashair Share will try to get it in a hacker's way, it still can't get master code in a high probability. As a result, you may need to browser hidden file: /SD_WLAN/CONFIG to find master code. Once you input and save the code, Flashair Share will save it using iOS encription system which is safe and reliable.

AP Mode

AP mode means Access Point Mode. FlashAir runs in AP mode by default. A wireless LAN device such as a PC and smartphone connects to the FlashAir as a wireless LAN client (station). So the device cannot access to the Internet when the device connected to the FlashAir.

Internet Thru Mode

When in Internet pass-thru mode, The FlashAir will behave like a router, bridging the connection between devices connected to it and a secondary (or "Parent") network.If the parent network is connected to the internet, your devices will have access to the FlashAir and the internet simultaniously.

Supported file format

It supports images and does not support video. The video made by camera will have a large size which will consume too much space. Besides, due to the resolution, it is not suitable to play the video directly on iPhone.

Missing thumbnail

When take a photo, camera will create a thumbnail automatically. Flashair share can't get the thumbnail if there isn't the thumbnail.

Offline mode

When you click thumbnail to see the large image, Flashair share will cache the large image so that it can save your time for next time. When you open Flashair share without connecting to FlashAir, the App will read the cached file--which is called offline mode.

The cache will not occupy iCloud space, but the local space. And you can delete the cache file in Settings.

Import to Album

When choose to import to album, Flashair Share will save the selected images into the album named of "Flashair Image Share". You'd better to authorize app to visit your Photo, or it can't save image successfully.

Import to Local

Flashair share will save selected images to your iPhone which can be visited in offline mode.

Import to PC

It needs iTunes and save the cached images to your PC by Doc Share function provided by Flashair share.
